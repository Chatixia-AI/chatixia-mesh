=====================================================

technologies, and all connections between them. Labels distinguish
control-plane traffic from data-plane traffic.

Connection Summary
-------------------
  Agent  <--> Sidecar      Unix socket IPC (JSON lines)
  Sidecar --> Registry      WebSocket (SDP offers/answers, ICE candidates)
  Sidecar <-> Sidecar      WebRTC DataChannel (P2P, DTLS encrypted)
  Hub     --> Registry      HTTP polling (agents, tasks, topology)
  Agent   --> Registry      HTTP heartbeat (POST /api/hub/heartbeat, ~15s)

Control plane  = Registry, WebSocket signaling, HTTP REST, Hub polling
Data plane     = WebRTC DataChannels between sidecars (direct P2P)
